请开发一个安卓app，用于背诵小学古诗词。

功能上：
- 请设法获取小学课本所有古诗词内容。
- 入口页
  - 最上方显示六个按钮，分别代表1-6年级，点击进入这个年级的诗词列表
  - 下方展示过去的学习记录，按学习顺序排倒序，展示每天学习过哪些诗词，是初学还是复习
  - 下方展示将来的复习计划，按复习日期排倒序，展示每天需要复习哪些诗词
- 诗词列表页：展示诗词的名称和作者，点击进入诗词详情页
- 诗词详情页：展示诗词内容。
  - 页面有个“标记学习”按钮，第一次点击则完成初学，以后点击则完成复习，每次点击都记下日期。每首诗词每天只能点击一次。学习当天是不能复习的。
  - 页面有个“隐藏原文”的按钮。如果进入的时候，已经“初学”过了，则隐藏原文，否则正常展示。
  - 页面下方展示过去的学习过程，包括初学于x月x日，复习于y月y日。
  - 页面下方有将来的复习计划，安排初次学习当天之后的1/4/8/15/30天后进行复习，比如今天是1月1日初学，那么后面就是1.2、1.4、1.8、1.15、1.30进行复习。某天如果复习过了，就一到上面的“过去的学习过程”部分。

技术上：
- 请采用流行/易维护/易调试的类h5方案。
- 最后打包成apk
  - gradle在 /opt/gradle/bin/gradle
  - 安卓开发工具在 /opt/android-sdk-linux
  - 安卓sdk在 ~/Android/Sdk
